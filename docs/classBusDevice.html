Detailed Description

template<typename addr_t, typename data_t>
class BusDevice< addr_t, data_t >

Abstract type used to represent a device which can be plugged in a bus. A bus device can be read and write to at specific address. It can only be "plugged in" to a bus using the same address and data types.
